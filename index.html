Amazon

I built a python PPG (photoplethysmogram) data generation model for fitness tracker emulation software that provides sensor data replay testing of Amazon Halo health technology algorithms. This significantly improved algorithm testing efficiency and accuracy because emulator can run tests faster than real time, multiple tests can be run in parallel, and signals are easily monitored throughout trials.

SuperWorld

During this summer I implemented new layer for virtual real estate map (desktop and mobile) that displays augmented reality artwork; the feature was showcased at the Flint Water Festival presenting an exclusive NFT art drop. I also integrated High Fidelity Spatial Audio API with virtual real estate map, which improved voice chat feature so that each sound source comes from a specific location in space instead of conventional VOIP audio. Finally, I Optimized mobile app by making Axios query filters and the rendering of virtual real estate data more efficient; this improved render times for map layer by 40%.

Duke Applied Machine Learning Group
